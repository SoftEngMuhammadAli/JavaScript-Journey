Learn Regular Expressions (Regex)

1. Matching a Simple Word

Regex: /hello/

This regex matches the exact word "hello".

2. Matching Digits

Regex: /\d+/

This regex matches one or more digits (0-9).

3. Matching an Email Address

Regex: /^[^\s@]+@[^\s@]+\.[^\s@]+$/

This regex matches a basic email format (e.g., user@example.com).

4. Matching a URL

Regex: /^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$/

This regex matches a basic URL format (e.g., https://example.com).

5. Matching a Phone Number

Regex: /^\d{10}$/

This regex matches a 10-digit phone number (e.g., 1234567890).
